CSS2 Specificatoin Release


This draft of the CSS2 specification is available in the following formats:

  • straight HTML on line,
  • a plain text file,
  • HTML as a gzip'ed tar file,
  • HTML as a zip file - this is a '.zip' file not an '.exe',
  • as well as a postscript, and a PDF file, thanks to html2ps written by Jan Kärrman.

We also plan to provide translations in other languages, although the English version provides the normative specification.
